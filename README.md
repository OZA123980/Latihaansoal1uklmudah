# Latihaansoal1uklmudah
Latihan ukl 1
Program ini berfungsi untuk menghitung total biaya pengiriman paket berdasarkan berat,jarak,dan volume paket. 
Cara kerjanya adalah: 
-Pengguna memasukkan berat,jarak,panjang,lebar,dan tinggi paket.
-Program menghitung volume = panjang×lebar×tinggi.
-Menentukan biaya per kg 
-Jika volume > 100 cm3,ditambah biaya Rp 50.000. 
-Program menampilkan total biaya pengiriman.
